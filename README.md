# myFlix API (Node.js / Express / MongoDB)

A REST API for a movie application. It provides endpoints to manage users, movies, and user favorites. Authentication is handled with JWT.

---

## Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- Passport (JWT)
- express-validator (input validation)

---

## Prerequisites
- Node.js (LTS recommended)
- npm
- MongoDB (local or MongoDB Atlas)
