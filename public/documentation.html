myFlix API Endpoints

Request URL HTTP Method Response
Get all movies /movies GET Text/JSON: list of all movies
Get movie by title /movies/:title GET Text/JSON: movie data by title
Get genre by name /genres/:name GET Text/JSON: genre description
Get director by name /directors/:name GET Text/JSON: director data
Register new user /users POST Text/JSON: user registered
Update user (username) /users/:username PUT Text/JSON: user updated
Add favorite movie /users/:username/movies/:movieId POST Text: movie added to favorites
Remove favorite movie /users/:username/movies/:movieId DELETE Text: movie removed from favorites
Deregister user /users/:username DELETE Text: user removed
